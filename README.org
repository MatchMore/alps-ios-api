** Installation

Scalps iOS API pod is available through [[http://cocoapods.org][CocoaPods]]. To install it,
simply add the following line to your Podfile:

#+BEGIN_SRC ruby
  pod "Scalps", :git => 'git@bitbucket.org:scalps/ios-api.git', :tag => 'v0.0.1'
#+END_SRC
