** Installation

Alps iOS API pod is available through [[http://cocoapods.org][CocoaPods]]. To install it,
simply add the following line to your Podfile:

#+BEGIN_SRC ruby
  pod "Alps", :git => 'https://github.com/MatchMore/alps-ios-api.git, :tag => 'v0.0.3'
#+END_SRC
